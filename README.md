# tob-helper — ToB 求职教练

> 不是给答案，是帮求职者自己看见答案。

覆盖 ToB 销售求职**全生命周期**的 Hermes Agent 技能。七阶段递进式辅导框架，每个阶段可独立使用，也可串联走完整流程。

---

## 核心哲学

| 原则 | 含义 |
|------|------|
| **引导者，不是代答者** | 举镜子让用户自己看见，不直接告诉他答案 |
| **提问 > 陈述** | 映射表/话术库是脑中的框架，嘴上问问题 |
| **让用户自己走到结论** | 他说「对的」比你替他说的效果好 10 倍 |

## 特色模块

### 3E. 跨行/零基础小白优势挖掘

针对非销售背景用户，通过**经历翻译三步法**将任何经历转化为销售语言。内置 13 行跨行业映射表，覆盖客服、运营、行政、技术、教师、财务、医生、工程、店长、设计、公安、物流、翻译等背景，每行配有**引导提问**而非预设答案。

### 4C. 被HR否定后的经历挖掘与重构

用户被 HR 拒绝后的完整应对流程：
1. **情绪接住** — 先做心理建设
2. **负面信号诊断** — 6 种拒绝理由对应引导方向
3. **深度挖掘 5 层链** — 事件→动作→困难→影响→自我翻译
4. **引导输出重构卡** — 给框架不给内容，让用户自己填充
5. **简历逐行提问** — 代替直接改写

## 部署方式

### Hermes Agent

自动加载：技能位于 `~/AppData/Local/hermes/skills/productivity/tob-helper/`，Hermes 在对话中检测触发词后自动加载。

### RedSkill / 小红书 SkillHub

通过 `skillhub-upload` CLI 上传至小红书云端：

```bash
skillhub-upload login --agent
skillhub-upload publish <path-to-skill> --agent --source original --tag "职场办公"
```
