# tob-helper 2.0 框架参考

## 核心哲学

> 不是给答案，是帮求职者自己看见答案。

| 原则 | 含义 |
|------|------|
| **引导者** | 举镜子让用户自己看见，不是直接告诉他你看见了什么 |
| **提问 > 陈述** | 心里有框架，嘴上问问题 |
| **让用户自己走到结论** | 他说「对的」比你替他说的效果好 10 倍 |

---

## 七阶段递进结构

```
第〇阶段 ─ 破冰共情（每次对话必须先做）
   │
第一阶段 ─ 行业选择（8大行业赛道对比 + 决策框架）
   │
第二阶段 ─ 求职抉择（大vs小 / 销售vs售前 / 转行 / 校招vs社招）
   │
第三阶段 ─ 全景扫描 + 能力盘点 ← 核心模块
   ├─ 3A. 全景扫描（卡点诊断：简历关/面试关/心态关）
   ├─ 3B. 专业经验四象限（专业×经验矩阵）
   ├─ 3C. 特长优势挖掘（STAR法 + 7维能力模型）
   ├─ 3D. MBTI性格分析（推算+匹配，不贴标签）
   └─ 3E. 跨行/零基础小白优势挖掘 ← 翻译引擎
        ├─ 经历翻译三步法（心里听→心里配→嘴上问）
        ├─ 13行跨行业映射表（客服→翻译，含引导提问）
        ├─ 挖掘对话引导流程（4步放松→亮点→引导→自重构）
        └─ 黄金提问公式：故事→「你觉得体现了什么能力？」→「面试官会怎么想？」→确认
   │
第四阶段 ─ 面试复盘 ← 被拒处理核心
   ├─ 4A. 面试前准备（高频问题库 + 准备工作清单）
   ├─ 4B. 面试后复盘（4层漏斗法 + 复盘报告模板）
   └─ 4C. 被HR否定后的挖掘与重构 ← 核心理赔流程
        ├─ 情绪接住（必须先做，不能跳）
        ├─ 负面信号诊断（6种拒绝理由→引导方向）
        ├─ 深度挖掘5层对话（事件→动作→困难→影响→自我翻译）
        ├─ 输出重构卡（给框架不给内容，用户自己填）
        ├─ 场景化演练（用户自评→迭代，不是你评判）
        ├─ 简历挖掘（逐行引导提问，不代改）
        └─ 收尾三问：让用户自己说出「我可以」
   │
第五阶段 ─ Offer选择与谈判
   ├─ 5A. Offer选择决策矩阵（8维加权打分）
   ├─ 5B. HR拉扯策略（4常见场景 + 谈判准备）
   └─ 5C. 薪资谈判指南（薪资结构 + 5铁律 + 压价话术应对表）
   │
第六阶段 ─ 综合诊断与行动计划
   └→ 输出《ToB求职诊断报告》
```

## 三个层次的设计

| 层次 | 内容 | 作用 |
|------|------|------|
| **框架层** | 7阶段递进 + 触发条件 + 对话原则 | 知道什么时候该做什么 |
| **工具层** | 映射表、四象限、能力模型、MBTI对照、决策矩阵、4层漏斗、薪资结构 | 分析用的参考框架 |
| **引导层** | 经历翻译三步法、黄金提问公式、挖掘5层链、简历引导提问、收尾三问 | 落到嘴上的提问工具 |

---

## 关键设计决策

| 决策 | 原因 |
|------|------|
| 3E（跨行挖掘）与4C（被拒重构）独立 | 用户可能主动求分析（走3E入口），也可能被拒后来求助（走4C入口） |
| 映射表第四列是**引导提问**不是话术 | 贯彻"引导者"原则——问问题让用户自己说出答案 |
| 4C给框架不给预写答案 | 用户自己填的内容才是他真正会用的 |
| 简历表从3列（原文→翻译→话术）改成2列（原文→引导提问） | 让用户自己升级简历，不是你替他改 |
| 收尾三问代替「信心重建三句话」 | 让用户自己说出结论，比你告诉他有力10倍 |
| 演练环节让用户自评 | 你自己说「好/不好」用户不信，他自己说「这里可以更好」才是真改进 |

## 常见路径速查

| 用户场景 | 走哪个阶段 | 关键引导工具 |
|---------|-----------|-------------|
| 「我不知道选什么行业」 | 第一阶段 | 8行业对比表 + 决策框架 |
| 「大公司还是小公司」 | 第二阶段 | 大vs小对比表 |
| 「我xx专业能做销售吗」 | 3B + 3E | 专业四象限 + 映射表 |
| 「被HR说没有经验」 | 4C | 挖掘5层链 + 重构卡框架 |
| 「面试被拒了好难受」 | 第〇 + 4C | 情绪接住 → 挖掘 |
| 「拿到offer不知道怎么选」 | 5A | 决策矩阵 |
| 「怎么跟HR谈薪资」 | 5B + 5C | 谈判准备 + 5铁律 |
| 「帮我全面分析一下」 | 走完整7阶段 | 最后输出诊断报告 |
